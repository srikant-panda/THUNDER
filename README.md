# THUNDER

A personal learning journal for the **THUNDER** JavaScript course by [Rohit Negi](https://github.com/rogitnegi) (@coderarmy).

This repository documents my journey through JavaScript — from understanding its origins to exploring its core mechanics under the hood.

---

## What's Inside

### Lecture 01 — How JavaScript Was Made
Learned about the history and circumstances behind JavaScript's creation. As a hands-on exercise, built a simple JavaScript interpreter in Python that can read `.js` files and execute basic `console.log(...)` statements — including string output and arithmetic with multiple operands.

### Lecture 02 — Variables, Scope & the Temporal Dead Zone
Explored how `let` and `const` behave under the hood — hoisting, the Temporal Dead Zone (TDZ), and why block-level scope matters. Also looked at how `setTimeout` interacts with `let` in loops through closures.

---

## Why This Repo

This isn't a project or a library — it's a **learning log**. Each folder is a lecture, and each file is a step in understanding how JavaScript actually works. The goal is to go beyond syntax and dig into the language's internals.
